Experience

 Research Experience

  • Intelligent Manipulation Lab, Lincoln Centre for Autonomous Systems
    Univeristy of Lincoln | Jul 2020 - Present
    • Performed point cloud image segmentation with cluster based methods to autonomously grasp object on a cluttered table with Dex-Net.

  • Robotics Research Group
    BITS Hyderabad | Feb 2019 - May 2020
    • Built a telepresence robot with teleoperation and autonomous modes of control. Employed ego-centric framework and formulated distance-area relation to track a person in autonomous mode. Analyzed kinematics of three-wheeled robots to develop a custom ROS teleoperation package.
    • Developed a shared autonomy framework through a web interface to increase accessibility and usability and carried out user studies to show that this approach increases the overall performance of the system.

 Work Experience

  • Technoyantra
    Pune | Jun 2020 - Jul 2020
    • Created HTML web interface that enables novice users to control autonomous mobile robots inside warehouses efficiently. Integrated various ROS libraries for carrying out the entire navigation process through the web.
    • Implemented human follower mode using YOLO algorithm and integrated it with safe teleoperation to avoid obstacles while following human.

  • APTRI Labs
    Ahmedabad | May 2019 - Jul 2019
    • Developed an open-source floor-assistant robot that can be controlled over the internet. Built the robot from scratch, assembled electronics, wrote shell scripts and employed the Lighttpd webserver.

  • Vinnan Labs
    Hyderabad | Oct 2018 - Feb 2019
    • Successfully applied the ROS Navigation stack on a mobile robot, used gmapping for SLAM and employed 3D mapping using Octomap. Used Dijkstra’s algorithm for path planning and AMCL for robot localization.

 Teaching Experience

  • Artificial Intelligence for Robotics, BITS F327
    BITS Hyderabad | Aug 2020 - Present
    • Assisting the faculty in designing the lab component of the course (weekly lab experiments) on using ROS and Gazebo simulations for implementing various AI algorithms.

Projects

  • Autonomous UAV framework for search and rescue mission
    Applied approximated Q-learning algorithm integrated with a PID position controller to locate a missing victim. Further, uniquely combined YOLO with Optical flow to track and follow the target victim.

  • Smart spoon
    Fabricated a 3D printed low-cost spoon for people with hand tremors. Used an inertial measurement unit(IMU) to sense tremors and applied a PID control algorithm to keep the spoon stable.

  • Biped-humanoid
    Analyzed and designed the robot on Solidworks, integrated the electronics and built the robot. Wrote a custom walking algorithm on Arduino and tested it on the robot.
