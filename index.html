Hi!

I am a senior in mechanical engineering at BITS Pilani, Hyderabad Campus, India. I am incredibly passionate about building robots and intelligent agents that move and interact with the world in useful ways.

Currently, I am a research intern at Intelligent Manipulation Lab, University of Lincoln, where I work on robotic manipulation and grasping. I have previously worked in the field of mobile robotics, telerobotics, human-robot interaction, aerial robotics and computer vision. I have now developed a research interest in Reinforcement Learning (RL). I am driven by the prospect that RL can serve as a general framework for solving intelligent decision-making problems in robotics.

In the past, I've had the good fortune of working closely with some wonderful researchers such as Prof. Arshad Javed and Prof. Paresh Saxena. I have also led the software team of Hyperloop India to participate in the Hyperloop pod competition organized by SpaceX.

Do reach out to me if you feel my work align with your personal/organizational goals. I am always on the lookout for meaningful opportunities and collaborations.

Past Projects

Biped-Humanoid Robot

Analyzed and designed a 8-dof biped humanoid robot on Solidworks, integrated the electronics and built the robot. Wrote a custom walking algorithm on Arduino and tested it on the robot. [code]

Autnomous Navigation of differential drive robot

Successfully applied the ROS Navigation stack, used gmapping for SLAM and employed 3D mapping using Octomap. Used Dijkstra’s and A* algorithm for path planning and Adaptive Monte Carlo Localization algorithm for robot localization. [Code]

  • ALFA: Developed open-source APTRI Labs Floor-Assistant (ALFA) robot that can be controlled over the internet. Built the robot from scratch, assembled electronics, wrote shell scripts and employed the Lighttpd webserver. [code]

  • Smart spoon: Fabricated a 3D printed low-cost spoon for people with hand tremors. Used an inertial measurement unit(IMU) to sense tremors and applied a PID control algorithm to keep the spoon stable.

  • Line Maze solver using OpenCV: Implemented multiple centroid tracking algorithm to follow lines and used the left-hand rule to solve the maze. [code]

  • Hexacopter: Successfully built a Hexarotor based on the pixhawk flight controller equipped with GPS and telemetry modules. [link]

  • Robotic arm: Created custom urdf model of a robot manipulator in gazebo and performed motion planning using Moveit and python scripts. [code]

  • Voice Controlled Car: Built a car that can be controlled using specific voice commands. It has about 50K+ views on Arudino.cc. [link]

  • Visual navigation for quadrotor: Implemented Monocular SLAM using PTAM (Parallel Tracking and Mapping) on a Gazebo simulation of a quadrotor. [code]

  • Radar using Processing 3: Object detection system that measures the distance from the surroundings and shows it on screen using the Processing 3 software. [link]

  • Mobile Controlled Car: Developed a mobile robot that can be controlled by a mobile phone using an android application. [link]

  • Secret Clap door lock: Built arduino based system that hears the pattern of claps and compares it with the saved pattern. If both matches, the lock opens. [link]

  • Gesture controlled car: Created a mobile robot that can be controlled by human hand gestures. Tilt of the hand determines the direction of the movement of the car. [link]
