Projects

teleop_keyboard_omni3

A Generic keyboard teleop for 3 wheeled omnidirectional robots. It lets you control a three-wheeled holonomic robot using keyboard keys. This package has been added to ROS package index (list of all ROS packages).

navros_pkg

Developed a ROS package for autonomous navigation of differential drive robots. Includes Gazebo simulation of a differential driven car that uses SLAM for mapping. Implemented Dijkstra's algorithm for path planning and localization is done using Monte Carlo localization algorithm.
